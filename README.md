# e-commerce-website
Subject 13: Project One: E-commerce Website
Este proyecto es un sitio web estático desarrollado con HTML5 y CSS3, diseñado para mostrar un catálogo de productos con imágenes.
El objetivo principal es demostrar habilidades de maquetación, diseño responsive y organización de archivos, como parte de mi portafolio frontend.
